
# Relatório Financeiro 

Projeto de aprendizagem com Power BI 

Dashboard interativa produzida em Power BI com dados fakes **Financials** fornecido pelo software

### objetivo 

 - aplicar recursos de layout
 -  segmentar dados
 -  indicadores 

---


## Como Executar

- Baixe o arquivo `Dashboard_finanrceiro.pbix` deste repositório.
- Abra no **Power BI Desktop**.
- Explore as páginas e navegações criadas.
